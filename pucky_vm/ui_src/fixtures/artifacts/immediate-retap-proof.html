Immediate re-tap proof

Purpose

This committed fixture page pairs with a longer audio clip so the inbox audio proof can verify the same-card pause race without depending on a live inbox card.

The success condition is simple: an immediate second tap during startup must still be honored and must return the tile to idle.

Why it exists

The original one-second WAV fixtures could finish naturally before the proof had enough time to distinguish a real queued pause from a false pass caused by clip completion.

This page is only here to keep the deterministic proof target self-contained in the shipped fixture bundle.
